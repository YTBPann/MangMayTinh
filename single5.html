Giới thiệu về Spring Boot

Spring Boot là một framework Java mạnh mẽ giúp phát triển ứng dụng web một cách nhanh chóng và dễ dàng. Nó cung cấp các công cụ hỗ trợ tự động cấu hình, giúp bạn không phải tốn thời gian cấu hình thủ công. Với Spring Boot, bạn có thể xây dựng ứng dụng web hiện đại và dễ dàng triển khai mà không cần phải lo lắng về các vấn đề phức tạp như cấu hình Spring.

Yêu cầu trước khi bắt đầu

  • Java Development Kit (JDK) 8 trở lên.
  • IDE như IntelliJ IDEA hoặc Eclipse.
  • Maven hoặc Gradle để quản lý phụ thuộc.
  • Kiến thức cơ bản về Java và Spring Framework.

Trước khi bắt đầu xây dựng ứng dụng web, hãy đảm bảo rằng bạn đã cài đặt đầy đủ các công cụ cần thiết như JDK và IDE. Bạn cũng cần làm quen với cách sử dụng Maven hoặc Gradle để quản lý các phụ thuộc trong dự án.

Bước 1: Tạo một dự án Spring Boot mới

Để bắt đầu, bạn có thể tạo một dự án Spring Boot mới bằng cách sử dụng Spring Initializr. Đây là công cụ trực tuyến giúp tạo ra một dự án Spring Boot cơ bản với cấu trúc thư mục và các phụ thuộc cần thiết. Truy cập Spring Initializr và thực hiện các bước sau:

  • Chọn Project là Maven hoặc Gradle (tuỳ thuộc vào sở thích).
  • Chọn Language là Java.
  • Chọn Spring Boot phiên bản mới nhất.
  • Chọn Dependencies là Web (Spring Web) và Thymeleaf (nếu muốn sử dụng giao diện người dùng).

Sau khi hoàn thành, nhấn "Generate" để tải xuống file .zip chứa dự án Spring Boot mới. Giải nén file và mở dự án trong IDE của bạn.

Bước 2: Tạo một Controller

Bây giờ chúng ta sẽ tạo một controller đơn giản để xử lý các yêu cầu HTTP. Tạo một lớp mới có tên là HomeController trong thư mục src/main/java/com/example/demo và thêm mã sau vào: 

                            
            package com.example.demo;
            
            import org.springframework.stereotype.Controller;
            import org.springframework.web.bind.annotation.GetMapping;
            
            @Controller
            public class HomeController {
                
                @GetMapping("/")
                public String home() {
                    return "index";
                }
            }

Trong ví dụ trên, chúng ta tạo một controller với một phương thức home() để trả về trang index.html khi truy cập vào URL gốc của ứng dụng.

Bước 3: Tạo View với Thymeleaf

Tiếp theo, chúng ta sẽ tạo một trang HTML để hiển thị cho người dùng. Trong thư mục src/main/resources/templates, tạo một file mới có tên là index.html và thêm mã sau vào: 

                            
            
            
            
                
                
                Ứng dụng Spring Boot
            
            
                
Chào mừng đến với ứng dụng Spring Boot!

                
Đây là trang chủ của ứng dụng web đơn giản của chúng ta.

Đây là một trang HTML đơn giản sử dụng Thymeleaf để hiển thị nội dung. Spring Boot sẽ tự động tìm kiếm các file .html trong thư mục templates.

Bước 4: Chạy ứng dụng

Để chạy ứng dụng, bạn chỉ cần chạy phương thức main trong lớp DemoApplication (được tạo sẵn khi bạn tạo dự án Spring Boot). Mở terminal trong IDE và nhập lệnh sau: 

                            
            mvn spring-boot:run

Sau khi ứng dụng khởi động, mở trình duyệt và truy cập http://localhost:8080. Bạn sẽ thấy trang chủ với thông báo "Chào mừng đến với ứng dụng Spring Boot!".

Bước 5: Thêm Model và kết nối với cơ sở dữ liệu

Tiếp theo, chúng ta sẽ thêm một model để quản lý dữ liệu người dùng. Tạo một lớp mới có tên User trong thư mục src/main/java/com/example/demo: 

                            
            package com.example.demo;
            
            import javax.persistence.Entity;
            import javax.persistence.Id;
            
            @Entity
            public class User {
                
                @Id
                private Long id;
                private String name;
                private String email;
            
                // Getters and setters
            }

Ở đây, chúng ta sử dụng JPA để kết nối với cơ sở dữ liệu và ánh xạ đối tượng User với bảng trong cơ sở dữ liệu. Bạn cần cấu hình kết nối cơ sở dữ liệu trong file application.properties.

Kết luận

Qua bài viết này, bạn đã biết cách xây dựng một ứng dụng web cơ bản với Java và Spring Boot. Chúng ta đã tạo dự án Spring Boot, tạo controller, view và thêm model để kết nối với cơ sở dữ liệu. Bạn có thể tiếp tục mở rộng ứng dụng này bằng cách thêm các tính năng như đăng nhập, đăng ký người dùng, hoặc quản lý các bài viết.

Chúc bạn thành công trong việc phát triển ứng dụng web với Java và Spring Boot!
