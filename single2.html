Sử dụng Java Collections Framework: Từ lý thuyết đến thực hành

Được viết bởi: [Tên tác giả]

Ngày đăng: [Ngày]

Giới thiệu về Java Collections Framework

Java Collections Framework (JCF) là một bộ các lớp và giao diện trong Java, cung cấp các cấu trúc dữ liệu hữu ích để lưu trữ và xử lý các đối tượng trong ứng dụng của bạn. Từ các danh sách (List), bộ sưu tập (Set), bản đồ (Map) cho đến các hàng đợi (Queue), JCF hỗ trợ nhiều kiểu cấu trúc dữ liệu khác nhau, giúp lập trình viên dễ dàng quản lý và thao tác với dữ liệu.

Trong bài viết này, chúng ta sẽ cùng tìm hiểu lý thuyết về các loại collections trong Java, sau đó sẽ đi vào các ví dụ thực tế về cách sử dụng chúng trong các ứng dụng Java.

Các loại Collections trong Java

Java Collections Framework cung cấp ba loại chính của collections:

  • List: Lưu trữ các phần tử theo thứ tự và cho phép trùng lặp.
  • Set: Lưu trữ các phần tử không trùng lặp.
  • Map: Lưu trữ cặp khóa-giá trị.
  • Queue: Quản lý hàng đợi của các phần tử, thường được sử dụng trong các bài toán xử lý theo thứ tự (FIFO).

1. Sử dụng List trong Java

List là một trong những collection phổ biến nhất trong Java. Một List lưu trữ các phần tử theo thứ tự và cho phép trùng lặp. Dưới đây là ví dụ về cách sử dụng ArrayList, một loại List phổ biến: 

                            
            import java.util.ArrayList;
            
            public class ListExample {
                public static void main(String[] args) {
                    ArrayList fruits = new ArrayList<>();
                    fruits.add("Táo");
                    fruits.add("Chuối");
                    fruits.add("Cam");
                    fruits.add("Táo");
            
                    // In ra danh sách
                    for (String fruit : fruits) {
                        System.out.println(fruit);
                    }
                }
            }

Trong ví dụ này, chúng ta sử dụng ArrayList để lưu trữ các loại trái cây. Bạn có thể thấy rằng ArrayList cho phép các phần tử trùng lặp, như trong trường hợp của "Táo" xuất hiện hai lần.

2. Sử dụng Set trong Java

Set là một collection không cho phép các phần tử trùng lặp. Ví dụ, khi thêm phần tử trùng lặp vào một HashSet, phần tử đó sẽ không được thêm vào. Dưới đây là ví dụ về cách sử dụng HashSet: 

                            
            import java.util.HashSet;
            
            public class SetExample {
                public static void main(String[] args) {
                    HashSet fruits = new HashSet<>();
                    fruits.add("Táo");
                    fruits.add("Chuối");
                    fruits.add("Cam");
                    fruits.add("Táo");  // Phần tử trùng lặp sẽ không được thêm vào
            
                    // In ra danh sách
                    for (String fruit : fruits) {
                        System.out.println(fruit);
                    }
                }
            }

Trong ví dụ này, chúng ta sử dụng HashSet để lưu trữ các loại trái cây. Mặc dù "Táo" được thêm hai lần, nhưng trong HashSet, chỉ có một "Táo" được giữ lại.

3. Sử dụng Map trong Java

Map là một collection lưu trữ các cặp khóa-giá trị. Mỗi khóa phải là duy nhất, nhưng giá trị có thể trùng lặp. Dưới đây là ví dụ về cách sử dụng HashMap: 

                            
            import java.util.HashMap;
            
            public class MapExample {
                public static void main(String[] args) {
                    HashMap ageMap = new HashMap<>();
                    ageMap.put("John", 25);
                    ageMap.put("Jane", 30);
                    ageMap.put("Alex", 22);
            
                    // In ra các cặp khóa-giá trị
                    for (String name : ageMap.keySet()) {
                        System.out.println(name + ": " + ageMap.get(name));
                    }
                }
            }

Trong ví dụ này, chúng ta sử dụng HashMap để lưu trữ tuổi của các người. Mỗi người (khóa) được liên kết với một giá trị tuổi (giá trị). Bạn có thể thấy cách lấy giá trị từ Map thông qua khóa.

4. Sử dụng Queue trong Java

Queue là một collection đặc biệt, thường được sử dụng trong các hệ thống yêu cầu xử lý theo thứ tự. Các phần tử trong Queue thường được xử lý theo quy tắc FIFO (First-In-First-Out). Dưới đây là ví dụ về cách sử dụng LinkedList làm Queue: 

                            
            import java.util.LinkedList;
            import java.util.Queue;
            
            public class QueueExample {
                public static void main(String[] args) {
                    Queue queue = new LinkedList<>();
                    queue.add("John");
                    queue.add("Jane");
                    queue.add("Alex");
            
                    // Xử lý các phần tử trong hàng đợi theo FIFO
                    while (!queue.isEmpty()) {
                        System.out.println("Xử lý: " + queue.poll());
                    }
                }
            }

Trong ví dụ này, chúng ta sử dụng LinkedList để làm hàng đợi. Các phần tử được xử lý theo thứ tự mà chúng được thêm vào (FIFO).

Kết luận

Java Collections Framework là một công cụ mạnh mẽ và cần thiết khi lập trình với Java. Với các loại collection như List, Set, Map, và Queue, bạn có thể dễ dàng xử lý các tập hợp dữ liệu trong các ứng dụng của mình. Việc hiểu rõ và sử dụng thành thạo các collections sẽ giúp bạn viết mã hiệu quả hơn, giảm thiểu sự phức tạp và tối ưu hóa hiệu suất của chương trình.
